DESIGN PRINCIPLES
══════════════════════════════════════════════════════════════════════════════

1. DARK BACKGROUNDS (#282c34, #21252b)
   - Reduces eye strain during long work sessions
   - Professional, modern appearance
   - Popular with developers (matches VS Code)

2. CONSISTENT TEXT (#abb2bf)
   - High contrast with dark background
   - Easy to read everywhere
   - Matches VS Code foreground color

3. BLUE ACCENTS (#61afef)
   - Primary interactive color
   - Stands out from gray background
   - Matches VS Code primary accent

4. SEMANTIC COLORS
   - Green (#98c379): Success, "Valid"
   - Red (#e06c75): Errors, "Delete"
   - Yellow (#e5c07b): Warnings, Info
   - These are intuitive to users

5. SUBTLE HOVER STATES (#3e4451)
   - Clear but not overwhelming
   - Provides feedback on interactivity
   - Subtle transition between states

6. 4px BORDER RADIUS
   - Modern, rounded corners
   - Not too rounded (professional)
   - Consistent across all elements

7. 6-8px PADDING
   - Comfortable spacing around elements
   - Not too cramped or loose
   - Professional typography

8. 1px BORDERS
   - Subtle element definition
   - Uses gray (#3e4451) consistently
   - Defines boundaries without clutter

QUICK INTEGRATION
══════════════════════════════════════════════════════════════════════════════

Option 1: Copy Complete File
  cp main_window_THEMED.py h2_plant/gui/ui/main_window.py
  (Complete main_window with theme baked in)

Option 2: Add to Existing Main Window
  from h2_plant.gui.themes.one_dark_pro_theme import get_stylesheet

  In __init__:
    self.setStyleSheet(get_stylesheet())

Option 3: Global Application Theme
  from h2_plant.gui.themes.one_dark_pro_theme import get_stylesheet

  In app creation:
    app = QApplication(sys.argv)
    app.setStyleSheet(get_stylesheet())
    window = PlantEditorWindow()


CUSTOMIZATION EXAMPLES
══════════════════════════════════════════════════════════════════════════════

Change Button Color:
  COLORS = {
      "blue": "#YOUR_HEX_CODE",  # In one_dark_pro_theme.py
      ...
  }

Change Background:
  COLORS = {
      "background": "#YOUR_HEX_CODE",
      ...
  }

Add Custom Widget Style:
  In get_stylesheet(), add:
  MyCustomWidget {
      background-color: #21252b;
      color: #abb2bf;
      border: 1px solid #3e4451;
  }

Create Light Version:
  Copy one_dark_pro_theme.py to light_theme.py
  Change COLORS to light palette
  Import light_theme instead
