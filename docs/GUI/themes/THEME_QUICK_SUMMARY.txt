## ğŸ”§ Customization

### Change a Color
Edit `one_dark_pro_theme.py`:
```python
COLORS = {
    "blue": "#YOUR_HEX_CODE",  # Change button color
    "background": "#YOUR_HEX", # Change background
    # ... etc
}
```

### Apply to Existing main_window.py
Add this to `__init__`:
```python
from h2_plant.gui.themes.one_dark_pro_theme import get_stylesheet

# After super().__init__():
self.setStyleSheet(get_stylesheet())
```

### Apply Globally
Add to your app launcher:
```python
from h2_plant.gui.themes.one_dark_pro_theme import get_stylesheet

app = QApplication(sys.argv)
app.setStyleSheet(get_stylesheet())
window = PlantEditorWindow()
```

## ğŸ†˜ Troubleshooting

### Theme not applying?
1. Check file path: `h2_plant/gui/themes/one_dark_pro_theme.py`
2. Verify import is correct
3. Restart application completely

### Text not readable?
1. Verify foreground color in COLORS dict
2. Check display contrast settings
3. Adjust color manually if needed

### Specific element not themed?
1. Add selector to stylesheet
2. Match widget class name exactly
3. Check QSS syntax

### Want different theme?
1. Create new `my_theme.py`
2. Define COLORS and get_stylesheet()
3. Import instead of one_dark_pro_theme
